# Flash Card Game (General Psychology)

A lightweight browser-based flash card game organized by chapter to help study college-level general psychology topics.

## Features

- Chapter selector with five major psychology units.
- Question/answer flash cards.
- Flip card interaction by clicking the card or using the toggle button.
- Previous/next navigation and progress tracking.
- Shuffle mode for randomized review.

## Run locally

Open `index.html` directly in a browser, or serve the folder with a static server:

```bash
python3 -m http.server 4173
```

Then visit `http://localhost:4173`.
